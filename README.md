# Sperm-quality-microbiome-2026
This is where all files and data pertaining to the work performed on the microbiome of higher and lower quality sperm will be held. 

Data was curated from Zymo Microbiomics and analyzed with QIIME2 using command-line interface. Analyses performed included observed richness, evenness, Shannon index, Simpson index, Bray-Curtis distance, and Jaccard distance.
